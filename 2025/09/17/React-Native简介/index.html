一、简介

1.React Native

React Native是Facebook在React.js Conf 2015上推出的开源框架，是React的一个原生扩展，它允许我们通过React语法来开发ios和Android原生应用

资源网站：

  • 官网：https://reactnative.dev/
  • Github: https://github.com/facebook/react-native
  • 中文网：https://www.react-native.cn/

4.React Native的优势（为什么学习RN）

  • 开发体验好：用统一的代码规范开发移动端程序，不用关注移动端的差异
  • 开发成本低：开发一次，可以生成Android和IOS两个系统上的APP
  • 学习成本低：只要掌握JavaScript和React，就可以进行进行移动端开发了
